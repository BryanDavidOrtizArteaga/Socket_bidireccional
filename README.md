# Socket_bidireccional
Este proyecto es un ejemplo sencillo de cómo crear un chat bidireccional.
Por medio de los programas:

- Servidor (server.py)
- Cliente (client.py)

El servidor y el cliente pueden enviarse mensajes mutuamente de forma alternada.
